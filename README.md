# Next.js Playground

A small **Next.js** project to learn and experiment with Next.js, Tailwind CSS, and TypeScript.

## Getting Started

1. Clone the repo:

```bash
git clone https://github.com/JonasDias10/nextjs-playground.git
cd nextjs-playground
```

2. Install dependencies:

```bash
yarn install
```

3. Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

* `dev` - Run development server
* `build` - Build for production
* `start` - Start production server
